# Michelangelo Documentation

Official documentation for [Michelangelo](https://michelangelo.land), the iOS app for vibe-coding native apps with AI.

Build, preview, and share Expo and React Native apps directly from your iPhone using natural language.

## Live docs

Visit [docs.michelangelo.land](https://docs.michelangelo.land).

## What's inside

This repository contains the source for our public documentation, built with [Mintlify](https://mintlify.com).

- `v2/` — Current documentation for the Michelangelo iOS app.
- `v1/` — Previous version of the documentation.
- `images/` — Static assets.
