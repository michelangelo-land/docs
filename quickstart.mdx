## Step-by-step guide

Welcome to this step-by-step guide on how to create an application using michelangelo.land.&#x20;
Each section is accompanied by screenshots to make the steps easier to understand.

<AccordionGroup>
  <Accordion title="Step 1: Log in to michelangelo.land" icon="right-to-bracket">
    1. Visit <a href="https://code.michelangelo.land/sign-in">code.michelangelo.land</a> 👀

    2. **Log in** or register to access the platform 🚀
       You can choose between **two sign-in options**: 	&#x9;

       * **Sign in with GitHub**: quickly sign in using your GitHub account.	&#x9;

       * **Continue with Email**: enter your email address and you will receive an **OTP code** via email. Manually enter the code received to confirm your login.
  </Accordion>

  <Accordion title="Step 2: Define your project" defaultOpen={false} icon="code">
    1. After logging in to [code.michelangelo.land](https://code.michelangelo.land) you can start developing right away.

       * **First time login**: when you log in for the first time, michelangelo.land automatically creates your first project, so you can start developing your first app right away without having to set up anything.

       * **Subsequent logins**: the platform will automatically load the last project you worked on, allowing you to pick up where you left off.

    2. **Development interface**
       When you access a project, you will see a screen with the following items:

       * **Development prompts**: you can write prompts to generate components, modify code, or add functionality to your app.

       * **Guide section**: shows the steps to create and test your app.

       * **QR Code**: allows you to view your app directly on your mobile device using Expo Go.

       * **Code section**: displays the project structure with the *`app`*&#x66;older already set up with two files inside: *`_layout.tsx`*&#x61;nd *`index.tsx`*.

    From here, you can start interacting with michelangelo.land to develop your application. 🚀
  </Accordion>

  <Accordion title="Step 3: View your App on Mobile" defaultOpen={false} icon="mobile-screen">
    1. After the project is generated, you will see a **QR Code** in the ‘**Guide**‘ **section**.

    2. **Download** the **Expo Go** app on your mobile device (if you haven't already done so).

    3. **Scan** the **QR Code** with your phone's camera or with the Expo Go app.

    4. The app will open on your device, allowing you to see the **result in real time 📱**.
  </Accordion>

  <Accordion title="Step 4: Explore and Edit the code" defaultOpen={false} icon="pen">
    1. In the ‘**Code**‘ **section** you can access the **project structure**.

    2. Navigate through the pre-generated files and folders, including:

       * *`_layout.tsx`*: manages navigation;

       * *`index.tsx`*: the home page of the app.

    3. Edit the code directly from the platform to **customize your app** via the&#x20;

       <a href="https://docs.michelangelo.land/introduction#quick-start">Development Prompt</a>

       &#x20; and you can enhance it through the following features:

       * <Icon size={20} color="white" iconType="light" icon="files" />

         &#x20;**Select multiple files** of your app at once via the ‘**+ Add context**’ button. This allows michelangelo.land to have more information and improve the accuracy of the requested changes.

       * &#x20;

         <Icon icon="paperclip" size={20} color="white" iconType="regular" />

         &#x20;**Attach images** via the attachment icon to allow michelangelo.land to replicate a more specific design for your app.

       * <Icon icon="pen" size={20} color="white" iconType="light" />

         &#x20;**Manually edit the code** directly in the michelangelo.land editor via the button with the pencil icon.
  </Accordion>

  <Accordion title="Step 5: Save and Version the Project on GitHub" defaultOpen={false} icon="github">
    You can **connect your GitHub** repository to version your code through the following steps:

    1. Click the ‘

       <Icon icon="github" size={20} color="white" iconType="regular" />

       &#x20;**Connect**‘ button to connect the project to a repository.

    2. Follow the three steps to complete the connection:

       * **Authorize** michelangelo.land to access your GitHub account.

       * **Install** michelangelo.land in your account.

       * **Select the repository** where you want to save the project.

    3. After connecting, each **change** will be **saved directly** to your repository.

    You can also **download your app code** directly via the download icon button <Icon icon="arrow-down-to-line" size={20} color="white" iconType="regular" />.&#x20;
  </Accordion>

  <Accordion title="Step 6: Application Deployment" defaultOpen={false} icon="share">
    1. Click the ‘**Deploy**‘ button to start publishing.

    2. Follow the step-by-step guide to:

       * Create a **Build**.

       * **Configure** any required **parameters**.

       * **Launch** your application online.

    3. Once the deployment is complete, you will receive a **link to access** your app.
  </Accordion>

  <Accordion title="Conclusion" defaultOpen={false} icon="party-horn">
    You just created your application with michelangelo.land! 🥳 Now you can continue to modify, improve and distribute it easily.

    If you need help, visit the <a href="https://docs.michelangelo.land/introduction#support">Support section</a> of the documentation or <a href="https://docs.michelangelo.land/introduction#stay-connected">join the community!</a>
  </Accordion>
</AccordionGroup>
