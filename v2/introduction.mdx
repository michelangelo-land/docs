## Overview

**Michelangelo** simplifies full-stack application development by combining advanced **AI**, automatic code generation, and integration with popular development ecosystems such as **Expo**, **React Native**, **GitHub** and **Supabase**. Whether you prefer developing from desktop, mobile, or wearable devices, Michelangelo provides a unified experience to create and distribute high-quality applications, wherever you are.
